===========================================
คู่มือการใช้งานและย้ายเครื่อง ANG PRO
===========================================

แอปพลิเคชัน ANG PRO ถูกออกแบบมาให้สามารถคัดลอก (Copy) โฟลเดอร์ไปเปิดใช้งานที่คอมพิวเตอร์เครื่องอื่นในร้านได้ทันที โดยผ่านไฟล์ "start-ang-pro.bat"

วิธีเตรียมตัวเพื่อย้ายโปรแกรมไปเครื่องอื่น (Export):
-------------------------------------------------
1. ปิดหน้าต่างสีดำ (Terminal) ทั้งหมดที่กำลังรันโปรแกรมอยู่บนเครื่องหลัก
2. ดับเบิลคลิกหรือคลิกขวาที่โฟลเดอร์ "auto-parts-app" เครื่องหลัก 
3. (สำคัญมาก*) โปรดทำการลบโฟลเดอร์ที่ชื่อ "node_modules" ทิ้งไปก่อนทำการย้าย
   - เหตุผล: โฟลเดอร์นี้มีขนาดใหญ่มาก (มากกว่า 300MB) และมีไฟล์ยิบย่อยเป็นแสนไฟล์ การก็อปปี้หรือ Zip จะใช้เวลานานมาก ไม่จำเป็นต้องย้ายไปด้วย เพราะเครื่องใหม่จะดาวน์โหลดใหม่เอง
4. ย่อขนาด (Zip) หรือ Copy โฟลเดอร์ "auto-parts-app" ไปใส่ใน Flash Drive โอนย้ายไปยังคอมพิวเตอร์เครื่องปลายทาง

วิธีติดตั้งที่คอมพิวเตอร์เครื่องใหม่ (Import):
-------------------------------------------------
1. ลากโฟลเดอร์ "auto-parts-app" จาก Flash Drive ไปวางไว้ที่หน้า Desktop หรือ ไดร์ฟ D: ตามต้องการ
2. **เครื่องปลายทางนั้น จะต้องมีโปรแกรม Node.js** ติดตั้งไว้ในเครื่องก่อนเสมอ 
   - ดาวน์โหลดฟรีได้ที่: https://nodejs.org/ (แนะนำให้เลือกปุ่ม LTS ที่แปลว่าเสถียรที่สุด)
   - ติดตั้งโปรแกรม Node.js แบบ Next ไปเรื่อยๆ จนจบ
3. เมื่อเปิดโฟลเดอร์แอปมาแล้ว ให้ดับเบิลคลิกที่ไฟล์ **"start-ang-pro.bat"**
4. หน้าต่างสีดำจะเด้งขึ้นมา 
   - หากเป็นการเปิดครั้งแรกสุด ระบบจะขึ้นว่ากำลังโหลดไลบรารี ให้รอประมาณ 1-3 นาทีขึ้นอยู่กับเน็ตครับ
   - หากเจอเบราว์เซอร์เด้งว่าไฟร์วอลล์บล็อก (Windows Defender Firewall) ให้กด "Allow access" (อนุญาตการเข้าถึง) ด้วยครับ
5. ระบบจะโหลดหน้าเว็บ "อังอะไหล่PRO" มาให้ใช้งานที่เว็บ "http://localhost:3000" อัตโนมัติ

ข้อควรระวัง:
- **ฐานข้อมูลของคุณ** (ข้อมูลยี่ห้อรถและรายการอะไหล่ที่คุณกรอกไว้ทั้งหมด) จะถูกเก็บไว้ในไฟล์ชื่อ `prisma/dev.db` 
- ดังนั้น ถ้าคุณตั้งใจย้ายเครื่องและอยากให้ "ข้อมูลอะไหล่เดิม" ตามไปด้วย ห้ามเผลอลบไฟล์ `dev.db` ในโฟลเดอร์ `prisma` เด็ดขาดครับ
